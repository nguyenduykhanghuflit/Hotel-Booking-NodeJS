# HOTEL BOOKING
---
* Member : Nguyen Duy Khang 
* Description :Build an hotel booking website using NodeJS
## Installation

Project requires [Node.js](https://nodejs.org/)  to run.

Install the dependencies and devDependencies and start the server.

```sh
cd hotel-booking
npm i
npm start
```
